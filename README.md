# 🏗️ Site LM Arquitetura

## Instruções de Uso

### 1. **Logo**
A imagem da logo foi fornecida. Você precisa:
- Salvar a imagem da logo como `logo.png` na mesma pasta do projeto
- A imagem será automaticamente exibida no site

### 2. **Arquivos do Projeto**
O site contém:
- `index.html` - Página principal com todas as seções
- `style.css` - Estilos com tema verde profissional
- `logo.png` - Sua logo (adicionar manualmente)

### 3. **Como Executar**
- Abra o arquivo `index.html` em um navegador web
- Ou use uma extensão como "Live Server" no VS Code para melhor experiência

### 4. **Seções do Site**

✅ **Header** - Navegação fixa com logo e menu
✅ **Hero** - Seção inicial com chamada para ação
✅ **Serviços** - 6 serviços principais apresentados em cards
✅ **Projetos** - Galeria de projetos recentes
✅ **Sobre** - Informações sobre a empresa
✅ **Contato** - Formulário de contato e informações
✅ **Footer** - Rodapé com copyright

### 5. **Customizações Recomendadas**
Substitua no HTML:
- `(11) 9999-9999` - Seu telefone
- `contato@lmarquitetura.com.br` - Seu email
- `São Paulo - SP, Brasil` - Sua localização
- Adicione mais detalhes na seção "Sobre"
- Atualize os projetos com informações reais

### 6. **Cores Utilizadas**
- Verde principal: `#4a7c2e`
- Verde claro: `#6ba551`
- Verde escuro: `#2d5016`
- Texto: Verde e cinza escuro
- Fundo: Branco e tons leves

### 7. **Design Responsivo**
O site se adapta perfeitamente para:
- 💻 Desktops
- 📱 Tablets
- 📲 Smartphones
